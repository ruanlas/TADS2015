• background-color: Define a cor de fundo de um elemento. Exemplo:
    • background-color: yellow
    • background-color: red
    • background-color: blue
  • background-image: É utilizado para definir uma imagem de fundo. Exemplo:
    • background-image: url("imagem.png")
  • background-repeat: Controla a repetição da imagem de fundo. Exemplo:
    • background-repeat: repeat-x => A imagem se repete na horizontal
    • background-repeat: repeat-y => A imagem se repete na vertical
    • background-repeat: repeat => A imagem se repete na vertical e na horizontal
    • background-repeat: no-repeat => A imagem não se repete
  • background-attachment: Define se a imagem será estática(fixa) ou se irá rolar com a página. Exemplo:
    • background-attachment: scroll => A imagem rola com a página
    • background-attachment: fixed => A imagem é fixa
  • background-position: Define a posição da imagem de fundo na tela. Padrão, ela será posicionada no canto superior esquerdo da tela. Exemplo:
    • background-position: top left => canto superior esquedo
    • background-position: center center => centro da tela
    • background-position: bottom right => canto inferior direito
      • ou
    • background-position: 100px 100px => define as coordenadas em pixels na tela
