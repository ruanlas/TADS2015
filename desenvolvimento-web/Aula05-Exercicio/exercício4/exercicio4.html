Pseudo-Elementos

Pseudo-elementos são utilizados para estilizar parte de algum tipo de elemento. Nos exemplos abaixo iremos ver nitidamente como isso ocorre:

    Alguns pseudo-elementos:
  • ::after => Insere algma coisa após o conteúdo de cada elemento
  • ::before => Insere algma coisa antes do conteúdo de cada elemento
  • ::first-letter => Seleciona a primeira letra de cada elemento
  • ::first-line => Seleciona a primeira linha de cada elemento
  • ::selection => Seleciona a parte de um elemento que é selecionada por um usuário
